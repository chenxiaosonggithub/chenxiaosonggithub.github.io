1 注册ChatGPT

我的chatgpt账号是在2023年初注册了，以下方法不确定现在是否可用。

在chatgpt网站点击“Try on ChatGPT Plus”按钮，用邮件箱注册账号（国内邮箱不确定是否可以注册，如果不能可以使用gmail注册），短信激活使用sms-activate提供的短信激活服务，用支付宝充值一美元左右，在网站左下角点击“OpenAI”，选择“美国（物理)”，购买就可以使用短信激活服务，好像有时效限制，所以要尽快使用。

以上步骤完成，就可以尽情使用chatgpt学习和工作，还有赚钱。

2 谷歌上网助手（Ghelper）

建议使用chrome浏览器的“谷歌上网助手”，可以在chrome的web store搜索“Ghelper”或“谷歌上网助手”安装插件，也可以在Ghelper网站上下载插件安装。购买VIP时，注意新注册的Ghelper账号无法使用支付宝支付，要把支付链接尾部替换成 options.html?/pay/1/alipay。如果无法访问某些网站，可以在“Select Server”中切换。

3 v2ray代理服务器

v2ray的github项目为v2ray-core。

安装说明请参考install.md。

配置文件参考VMess-Websocket。

如果你在华为的HarmonyOS NEXT的”卓易通“中安装的v2ray，需要把“跌由设置“中的”绕过局域网IP“取消勾选，“绕过局域网域名”建议也取消勾选（但不确定有没影响哈），然后在“卓易通”和“出境易”中都能访问外网了。注意要先打开“卓易通”中的“搜应用”，才能在“出境易”中的浏览器中访问外网。

3.1 v2ray服务器安装与配置

可以使用以下命令安装:

bash <(curl https://raw.githubusercontent.com/v2fly/fhs-install-v2ray/master/install-release.sh)

也可以在项目的Releases界面选择一个版本下载Linux的安装包, 如v2ray-linux-64.zip-v5.4.1。

将/usr/local/etc/v2ray/config.json的内容替换成config_server.json中的内容，并修改以下内容

...
 "port": 12345, // 如果出现突然无法访问或不稳定，可以尝试修改端口
...
 "id": "12345678-1234-1234-1234-123456789012",      // 可自定义id
 "alterId": 64 // 客户端的值 <= 服务端的值, 两端都为64时无法使用(原因待分析)
...

修改完配置文件后，需要重启v2ray服务:

sudo systemctl restart v2ray

如果启动v2ray服务时报错，检查 v2ray.service文件:

[Service]
User=sonvhi # 修改成当前用户名
...

3.2 linux系统客户端安装与配置

可以使用以下命令安装:

bash <(curl https://raw.githubusercontent.com/v2fly/fhs-install-v2ray/master/install-release.sh)

也可以在项目的Releases界面选择一个版本下载Linux的安装包, 如v2ray-linux-64.zip-v5.4.1。

将/usr/local/etc/config.json的内容替换成config_client.json中的内容，并修改以下内容:

...
        "address": "请填写服务器ip", // 服务器地址，请修改为你自己的服务器 ip 或域名
        "port": 12345,  // 服务器端口
        "users": [{ 
            "id": "12345678-1234-1234-1234-123456789012",   // 与服务器id一样
...

修改完配置文件后，需要重启v2ray服务:

sudo systemctl restart v2ray

打开 设置 -> Network -> Manual -> HTTP proxy / HTTPS Proxy 127.0.0.1 1081

3.3 macOS系统客户端安装与配置

在项目的Releases界面选择一个版本下载macOS的安装包, 如v2ray-macos-arm64-v8a.zip-v5.4.1。

解压后将config.json的内容替换成config_client.json中的内容，并修改以下内容:

...
 "port": 12345, // 如果出现突然无法访问或不稳定，可以尝试修改端口
...
 "id": "12345678-1234-1234-1234-123456789012",      // 可自定义id
 "alterId": 64 // 客户端的值 <= 服务端的值, 两端都为64时无法使用(原因待分析)
...

修改完配置文件后，打开terminal，在解压的文件夹下执行以下命令:

./v2ray run

配置所连接网络的代理Server: localhost, Port: 1081。

3.5 Windows系统客户端安装与配置

在项目的Releases界面选择一个版本下载Windows的安装包, 如v2ray-windows-64-v8a.zip-v5.4.1

解压后将config.json的内容替换成config_client.json中的内容，并修改以下内容:

...
 "port": 12345, // 如果出现突然无法访问或不稳定，可以尝试修改端口
...
 "id": "12345678-1234-1234-1234-123456789012",      // 可自定义id
 "alterId": 64 // 客户端的值 <= 服务端的值, 两端都为64时无法使用(原因待分析)
...

修改完配置文件后，打开windows的cmd程序，在解压的文件夹下执行以下命令:

.\v2ray run

然后配置网络的http代理 localhost:1081。
