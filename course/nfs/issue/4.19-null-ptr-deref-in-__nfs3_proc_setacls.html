x24没有被改变，所以崩溃时x24寄存器中存的就是struct dentry对象的值:

struct dentry fffff62fe622b130 # x24寄存器的值
struct dentry {
  ...
  d_inode = 0x0,
  ...
}

这也从另一个角度证明了nfs3_proc_setacls()和__nfs3_proc_setacls()函数中传入的struct inode为空指针。
