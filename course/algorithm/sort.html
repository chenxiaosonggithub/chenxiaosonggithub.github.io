4 leetcode 217. 存在重复元素

  • 点击这里查看题目

这个题目主要是熟悉一下c语言快速排序的库函数qsort()的用法，

c语言实现:

int cmp(const void *a, const void *b)
{
    return *(int *)a - *(int *)b;
}

bool containsDuplicate(int* nums, int numsSize) {
    qsort(nums, numsSize, sizeof(int), cmp);
    for (int i = 1; i < numsSize; i++) {
        if (nums[i] == nums[i - 1])
            return true;
    }
    return false;
}
