键盘配置

  • 1 Linux
  • 2 macOS
  • 3 windows autohotkey
  • 4 hhkb

这里记录一下我在各个平台上使用的键盘配置，以及我使用的独立键盘。

1 Linux

xmodmap是Linux桌面系统用于更改键位分布的软件，具体查看《GNU/Linux配置文件》中有关xmodmap配置相关的章节。

2 macOS

Karabiner-Elements, 要开启权限Privacy & Security -> Input Monitoring, 添加karabiner_observer、karabiner_grabber、Karabiner-Elements、Karabiner-EventViewer。有时权限还是有问题，需要通过先卸载软件再安装来解决。

Complex Modifications复杂的映射要修改文件/Users/sonvhi/.config/karabiner/karabiner.json。

查看按键名称可以使用karabiner-EventViewer查看。

默认按caps_lock键会切换输入法，设置 Keybord -> Text input -> Input Sources -> Edit -> All Input Sources -> 关闭Use the Caps Lock key to switch to and from ABC

默认按caps_lock键会切换输入法，设置 Keybord -> Press fn(是一个地球图标) key to 选 Do Nothing。

我使用的配置文件: karabiner.json，放到macOS的~/.config/karabiner/karabiner.json路径下。

3 windows autohotkey

windows上更改键位可以使用autohotkey，有两个版本:

  • 按键列表v1
  • 按键列表v2

v1版本的配置: autohotkey-v1.1.ahk。

4 hhkb

我用的键盘是HHKB，另外还买了一个YDKB的控制模块。

更改键位方法: 访问ydkb.io网站，左上角选择“HHKB BLE”，然后根据右下方“HHKB BLE刷机说明”里的文档进行操作。

蓝牙出问题，可以尝试清除配对信息按键 LShift+RShift+LCtrl+R。
