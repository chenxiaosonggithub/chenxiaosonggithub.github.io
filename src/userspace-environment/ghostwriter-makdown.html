ghostwriter: дёҖж¬ҫmakdownзј–иҫ‘еҷЁ

  • 1 LinuxдёӢжәҗз Ғе®үиЈ…
  • 2 йў„и§Ҳз•ҢйқўжҳҫзӨәзјәйҷ·

еҶҷж–ҮжЎЈпјҢжңҖзҲҪзҡ„ж јејҸе°ұжҳҜmarkdownгҖӮ

жӣҫз»Ҹе°қиҜ•иҝҮеҮ ж¬ҫиҮӘз”ұејҖжәҗзҡ„markdownзј–иҫ‘еҷЁпјҢдҪҶйғҪдёҚеӨҹзҗҶжғіпјҢдёҚжҳҜеҠҹиғҪдёҚе®Ңе–„е°ұжҳҜзјәйҷ·еӨҡгҖӮ

зҺ°еңЁжҲ‘жҳҜзӣҙжҺҘеңЁиҮӘе·ұзҡ„дёӘдәәзҪ‘з«ҷдёҠжҹҘзңӢжңҖз»Ҳзҡ„жҳҫзӨәж•ҲжһңпјҢе…·дҪ“еҸҜд»ҘзңӢгҖҠеҰӮдҪ•жӢҘжңүдёӘдәәеҹҹеҗҚзҡ„зҪ‘з«ҷе’ҢйӮ®з®ұгҖӢгҖӮ

е°қиҜ•зҡ„е…¶дёӯдёҖж¬ҫзј–иҫ‘еҷЁе°ұжҳҜKDEзҡ„ghostwriterпјҢиҝҷдёӘmarkdownзј–иҫ‘еҷЁеҠҹиғҪжҢәе®Ңе–„пјҢдҪҶзјәйҷ·д№ҹдёҚе°‘гҖӮ

2021е№ҙж—¶жҲ‘и§ЈеҶідәҶдёҖдёӘдёҘйҮҚзҡ„йў„и§Ҳз•ҢйқўжҳҫзӨәзјәйҷ·Chinese preview in HTML is complete now. Update README.md.гҖӮ

еҰӮжһңеҗ„дҪҚжңӢеҸӢжғіиҮӘе·ұдҝ®ж”№жҲ–еўһеҠ еҠҹиғҪпјҢеҸҜд»ҘдҪҝз”Ёgithubжәҗз Ғе®үиЈ…гҖӮ

1 LinuxдёӢжәҗз Ғе®үиЈ…

ghostwriterжҳҜи·Ёе№іеҸ°зҡ„пјҢеңЁLinuxгҖҒmacOSгҖҒWindowsдёӢйғҪиғҪиҝҗиЎҢпјҢиҝҷйҮҢжҲ‘еҸӘд»Ӣз»ҚFedoraпјҲLinuxпјүдёӢзҡ„зј–иҜ‘иҝҗиЎҢпјҢе…¶д»–е№іеҸ°иҜ·еҸӮиҖғREADME.mdгҖӮ

е…Ҳе®үиЈ…FedoraдёӢзј–иҜ‘жүҖйңҖзҡ„иҪҜд»¶:

sudo dnf install qt-devel qt5-qtbase-devel qt5-qtsvg-devel qt5-qtmultimedia-devel qt5-qtwebengine-devel hunspell-devel qt5-linguist -y

еңЁжәҗз Ғзӣ®еҪ•дёӢзј–иҜ‘:

qmake-qt5
make
# е®үиЈ…е‘Ҫд»ӨеҸҜдёҚжү§иЎҢ
# make install

иҝҗиЎҢиҪҜд»¶:

cd build/release/
./ghostwriter

иҝҷдёӘиҪҜд»¶жӣҫз»Ҹз”ЁзқҖж„ҹи§үиҝҳиЎҢгҖӮ

2 йў„и§Ҳз•ҢйқўжҳҫзӨәзјәйҷ·

2021е№ҙ6жңҲеҲҡз”Ёghostwriterж—¶пјҢдёҖдёӘеҫҲжҳҺжҳҫзҡ„йў„и§Ҳз•ҢйқўжҳҫзӨәзјәйҷ·ж‘ҶеңЁжҲ‘йқўеүҚ: йў„и§Ҳз•Ңйқўдёӯж–ҮжҳҫзӨәдёҚе®Ңж•ҙпјҢеҲҡејҖе§Ӣжғіж”ҫејғиҝҷдёӘиҪҜд»¶пјҢдҪҶеҸҲи§үеҫ—е…¶д»–еҠҹиғҪжҢәе®Ңе–„зҡ„пјҢжңҖеҗҺиҝҳжҳҜеҶіе®ҡиҮӘе·ұи§ЈеҶізјәйҷ·гҖӮ

зҺ°еңЁжҠҠиҝҷдёӘзјәйҷ·д»Ӣз»ҚдёҖдёӢпјҢд№ҹеёҢжңӣеҗ„дҪҚжңӢеҸӢиғҪе’ҢжҲ‘дёҖиө·е®Ңе–„ghostwriterгҖӮ

еҜјиҮҙиҝҷдёӘзјәйҷ·зҡ„commit:

// src/cmarkgfmapi.cpp
-    cmark_parser_feed(parser, text.toLatin1().data(), text.length());
+    cmark_parser_feed(parser, text.toUtf8().data(), text.length());

е®ҡдҪҚеҲ°иҝҷдёӘеҺҹеӣ еҗҺпјҢжҲ‘дҝ®ж”№жҲҗеҰӮдёӢд»Јз Ғ:

// src/cmarkgfmapi.cpp
cmark_parser_feed(parser, text.toLatin1().data(), text.toLocal8Bit().length());

дёӯж–ҮжҳҜUnicodeзј–з ҒпјҢдёҖдёӘжұүеӯ—еҚ 2дёӘеӯ—иҠӮпјҢиҖҢQString.length()дёҖдёӘжұүеӯ—еҸӘеҪ“жҲҗдёҖдёӘеӯ—иҠӮпјҢйңҖиҰҒдҝ®ж”№жҲҗQString.toLocal8Bit().length()гҖӮ

жҲ‘зҡ„дҝ®ж”№ж–№жЎҲжҳҜChinese preview in HTML is complete now. Update README.md.пјҢlioneieжҳҜжҲ‘жӣҫз»Ҹзҡ„зӯҫеҗҚгҖӮ

дҪ жІЎзңӢй”ҷпјҢе°ұжҳҜиҝҷд№Ҳе®№жҳ“и§ЈеҶіпјҢеҸӮдёҺеҲ°иҮӘз”ұиҪҜд»¶е°ұжҳҜиҝҷд№Ҳз®ҖеҚ•пјҢеҸӮдёҺеҲ°ејҖжәҗиҪҜд»¶е°ұжҳҜиҝҷд№Ҳз®ҖеҚ•гҖӮ
